# Library Management System using Spring Boot

## 📌 Project Description
This is a Spring Boot based Library Management System developed to manage books, users, and issue/return operations.

## 🚀 Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## ⚙ Features
- Add Books
- View Books
- Issue Book
- Return Book
- Manage Users

## ▶ How to Run
1. Clone the repository
2. Open in IntelliJ / STS
3. Configure MySQL database
4. Run the Spring Boot application
